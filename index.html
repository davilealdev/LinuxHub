Ricing/Customização

  • Hidamari

    Wallpaper animado (muda o visual da área de trabalho).

  • Komorebi

    Outra opção de wallpaper animado, mais customizável.

  • mpvpaper

    Wallpaper em vídeo usando MPV (mais leve).

  • Variety

    Troca automática de wallpapers.

  • Conky

    Exibe informações do sistema na tela (CPU, RAM, rede).

  • KDE Plasma

    Ambiente extremamente customizável (temas, widgets, efeitos).

  • Plank

    Dock simples e leve estilo macOS.

  • Latte Dock

    Dock avançado com animações (para KDE).

    • ⚙️ NÃO são exatamente estilização (mas ajudam no setup)

  • Ulauncher

    Launcher rápido para abrir apps e executar comandos.

  • Rofi

    Launcher e seletor de janelas altamente customizável.

  • i3

    Gerenciador de janelas automático focado em produtividade.

  • Hyprland

    Gerenciador moderno com animações e controle avançado de janelas.

Ferramentas de Desenvolvimento

  • Visual Studio Code

    Editor de código leve com suporte a extensões.

  • IntelliJ IDEA

    IDE completa para Java e outras linguagens.

  • PyCharm

    IDE focada em desenvolvimento Python.

  • Eclipse

    IDE tradicional com suporte a várias linguagens.

  • Neovim

    Editor de código no terminal, rápido e customizável.

  • Vim

    Editor clássico no terminal, altamente eficiente.

  • Git

    Controle de versão para gerenciar código.

  • GitHub Desktop

    Interface gráfica para usar Git de forma simples.

  • Docker

    Cria ambientes isolados para aplicações.

  • Postman

    Testa APIs com interface amigável.

  • Insomnia

    Alternativa leve para testes de API.

  • Bash

    Shell padrão para automação e scripts no Linux.

    Linguagens de Programação

  • Python

    Linguagem versátil para automação, web, dados e IA.

  • Node.js (JavaScript)

    Runtime para executar JavaScript fora do navegador.

  • Java (OpenJDK)

    Plataforma para aplicações robustas e multiplataforma.

  • C / C++ (GCC)

    Compilador para linguagens de baixo nível e alto desempenho.

  • Rust

    Linguagem moderna focada em segurança e performance.

  • Go (Golang)

    Linguagem simples e rápida para sistemas e backend.

  • PHP

    Linguagem popular para desenvolvimento web.

  • Ruby

    Linguagem elegante muito usada com Ruby on Rails.

  • C# (.NET)

    Plataforma da Microsoft com suporte multiplataforma.

  • Scala

    Linguagem funcional que roda na JVM.
